# excel-trade-crypto-backend

Excel-Trade-Crypto 本地后端服务

已封装币安交易所现货、期货、期权REST API&websocket

## 前端仓库
[excel-trade-crypto](https://github.com/alexischiang/excel-trade-crypto)


## 使用方法
- 输入Api Key&secret 
```bash
在./index.js line 11处填入自己的api key和secret
```

- macOS
```bash
npm i 
nodemon ./index.js
```

- windows
```bash
npm i 
node ./index.js
```
